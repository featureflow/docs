Jira Integration

note

This integration is for Jira Cloud only.

With Featureflow's Jira integration, you can view and link feature flags directly from Jira's Development Panel. Any Jira issue IDs linked to your feature flags will automatically sync to appear in Jira.

Installation​

1. Generate Integration Token in Featureflow​

Navigate to Administration → Integrations and click Add To Jira.

Click Generate Token to create a secure integration token.

Copy the token — you won't be able to see it again.

2. Install the Featureflow App in Jira​

In Jira, click the Settings cog and select Marketplace apps.

Search for "Featureflow" and click Get it now.

3. Connect Featureflow to Jira​

In Jira Settings, go to Manage apps and click FeatureFlow Configuration.

Paste your integration token and click Connect to FeatureFlow.

Once connected, you'll see a success message.

Linking Features to Jira Issues​

In Featureflow, edit any feature and add Jira issue keys (e.g., PROJ-123) to the Jira Issues field. These will automatically appear in Jira's Development Panel for that issue.

Uninstalling​

To remove the integration:

  1. In Jira: Go to Manage apps → FeatureFlow Configuration and click Disconnect
  2. In Featureflow: Navigate to Administration → Integrations and click the trash icon next to the Jira integration
