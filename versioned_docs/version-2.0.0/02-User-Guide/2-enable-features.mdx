# Enable and Disable Features

Features have targeting rules which can be enabled or disabled in a given environment.

By default a feature can be 'Off' or 'On' - this is a simple boolean feature. To enable and disable your feature:

Using the main menu, navigate to your desired project and feature

Click the feature On/Off toggle agains the environment you wish to enable or diosable

If it is a production environment, you will get a confirmation warning
