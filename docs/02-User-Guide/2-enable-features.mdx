# Enable and Disable Features

Features have targeting rules which can be enabled or disabled in a given environment.

By default a feature can be 'Off' or 'On' - this is a simple boolean feature. To enable and disable your feature:

Using the main menu, navigate to your desired project and environment

Click the feature On/Off toggle in the features list

Confirm at the prompt
